# Python GUI Calculator

## Overview
A fully functional calculator built using Python's Tkinter library. This program supports basic arithmetic operations (addition, subtraction, multiplication, division), percentages, square roots, and positive/negative toggling. It features a clean and intuitive graphical interface that mimics a standard calculator.

## Technologies Used
Python • Tkinter • GUI Design

## Features
- Perform basic arithmetic operations: +, -, ×, ÷
- Percentage calculations
- Square root calculations
- Toggle positive/negative numbers
- Clear all entries (AC button)
- Graphical user interface using Tkinter
- Centered and responsive window layout

## How to Run
1.Make sure Python 3 is installed on your computer.

2.Install the required library (Tkinter is included by default with Python):

```bash
# Tkinter usually comes with Python, but if missing, install it:
pip install tk
```
3.Open the terminal or command prompt, choose a directory, and create a folder for the project.

4.Go to the project directory.

5.Clone the project from GitHub:
```bash
git clone https://github.com/Pythonista-dev-cyber/Python-GUI-Calculator.git
cd Python-GUI-Calculator
```

6.open VS code with code.

7.run the file.
